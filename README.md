# Flask

## create a envoriament
```python -m venv <your_folder>```
```python -m venv env```

## activate the env
```<your_folder>\scripts\activate```
```env\scripts\activate```

## deactivate 
```deactivate```

## create new database, in python
```from app import db```
```db.create_all()```


## start your application
$env:FLASK_APP="/app.py"

## set the enviroment
$env:FLASK_ENV="development"
